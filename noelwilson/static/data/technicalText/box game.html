A simple web game using
  • Python
  • Django
  • MySQL
  • Javascript
  • JQuery
  • Html
  • CSS
Dynamically creates Html pages using a mixture of Django templates and JQuery 
will load a game where the user has to keep his mouse over a randomly moving 
box while his time is recorded. If a high score is attained they will be able 
to enter it with a name and it will be stored on a database and displayed on 
the website.
